# Blinkit-Merchandising-Outlet-Analytics
## Project Overview

This project presents an end-to-end business intelligence dashboard developed using Power BI to analyze Blinkit’s sales, outlet performance, merchandising efficiency, and item profitability.The dashboard enables stakeholders to understand sales trends, outlet behavior, product performance, and operational insights across different outlet sizes, locations, and market tiers, supporting data-driven decision-making.
## Business Objectives

• Analyze overall sales performance across outlets and item categories  
• Identify high- and low-performing outlet types, sizes, and locations  
• Evaluate merchandising efficiency using item visibility and ratings    
• Understand customer purchasing patterns using item attributes  
• Compare revenue and volume contribution across market tiers  
## Dashboard Overview

### Page 1: Merchandising and Outlet Performance  
This dashboard page focuses on sales distribution, product mix, and outlet-level characteristics.

### Key KPIs

• Total Sales: $1.20M  
• Average Sales: $141  
• Number of Items: 8,523  
• Average Rating: 3.9  

### Analysis Covered

• Sales trend analysis by outlet establishment year  
• Sales contribution by outlet size (Small, Medium, High)  
• Revenue distribution by outlet location tier  
• Product performance analysis by item type  
• Impact of fat content (Low Fat vs Regular) on total sales  
• Outlet-wise comparison of sales, item count, ratings, and item visibility  

### Page 2: Outlet Operations and Item Profitability  
This dashboard page focuses on operational efficiency and profitability at the outlet and item level.

### Key KPIs

• Total Sales: $1.20M  
• Outlet Establishments: Approximately 2,000  
• Average Item Visibility: 0.07  
• Total Items: 8,523  

### Analysis Covered

• Revenue and volume comparison by outlet market tier  
• Sales contribution by outlet type and outlet size  
• Outlet-level performance analysis using outlet identifiers  
• Relationship between average item visibility and total sales  
• Identification of high-performing and underperforming product categories  
## Key Insights

• Tier 3 outlets contribute the highest revenue and item volume  
• Medium-sized outlets show consistent performance across outlet types  
• Fruits and Vegetables and Snack Foods are the top revenue-generating categories  
• Regular fat content items generate higher total sales compared to low-fat items  
• Higher item visibility does not always guarantee higher revenue, indicating optimization opportunities

• Microsoft Excel / CSV as data sources

• Download the Power BI (.pbix) file  
• Open the file using Power BI Desktop  
• Use slicers to filter data by outlet location tier, outlet size, item type, and item fat content  
## Conclusion

This project demonstrates the ability to transform raw retail data into meaningful business insights using Power BI. The dashboards are designed to support decision-makers in evaluating performance, identifying trends, and optimizing merchandising and outlet strategies.
